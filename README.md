# Bio-Pod

Public marketing and brand content for Bio-Pod — a pre-measured, biodegradable organic fertilizer pod. Drop it in the soil, water as normal. 100% organic, zero synthetic chemicals, zero plastic waste.

## What's here
- `website/` — marketing site content
- `product-info/` — public-safe product descriptions
- `media/` — press releases and approved photos
- `brand/` — logo, colors, approved messaging

## Contributing
This repository is **public**. Every change goes through a Pull Request, and the **Commit Guardian** check must pass before merging — it blocks anything that looks like financial or other internal data. When in doubt, it stays out. See `CLAUDE.md`.
